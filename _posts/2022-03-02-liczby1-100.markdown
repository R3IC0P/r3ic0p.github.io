Zadanie wypisz liczby podzielne przez 5 z przedzia≈Çu (1-100)
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Liczby podzielne przez 5 - Walukiewicz</title>
</head>
<body>
  <script>
    for (let i=1; i<=100; i++) {
      if (i%5 === 0) {
        document.write(`${i}<br>`)
      }
    }
    // Walukiewicz Lukasz
  </script>
</body>
</html>
```
